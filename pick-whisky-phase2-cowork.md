## UI仕様

### ボタン

```
今日の運勢で選ぶ  ✦
サブテキスト: 「条件なし。AIの直感に任せる。」
```

- ボタンカラー: `--surface2` 背景 + `--accent` ボーダー（メインCTAとは差別化）
- タップすると即APIコール → ローディング → 結果モーダル表示

### 結果モーダル

```
今日のあなたの1本

[ボトル画像]
[銘柄名]
[タグ: 産地 / フレーバー / 価格帯]
[一言コメント（1文）]

[Amazon →]  [楽天 →]

[もう1本引く]  [閉じる]
```

- モーダル背景: `rgba(0,0,0,0.7)` オーバーレイ
- モーダル本体: `--surface2` + `--accent` ボーダー
- 「もう1本引く」で再度APIコール

### シェアボタン（任意・推奨）

```
X（Twitter）でシェア
```

シェアテキスト例:
```
今日のウイスキーは「グレンフィディック 12年」らしい。
AIのおみくじで引いた。 #SOWHAT #ウイスキー
https://pick.sowhat.monster/whisky
```

## APIプロンプト仕様

### システムプロンプト

```
あなたはウイスキーと焼酎の専門家です。
今日のおみくじとして、ランダムに1本レコメンドしてください。
毎回異なる銘柄を返すようにしてください。

必ずJSON形式のみで返してください。

{
  "name": "銘柄名（正式名称）",
  "tags": ["産地", "フレーバー特徴", "価格帯"],
  "comment": "今日この1本を引いたあなたへの一言（1文・軽いトーンで）",
  "amazonKeyword": "Amazon検索用キーワード",
  "rakutenKeyword": "楽天検索用キーワード"
}
```

### ユーザープロンプト

```
今日（{今日の日付}）のおみくじとして、ウイスキーまたは焼酎を1本ランダムにレコメンドしてください。
```

- 日付をシードとして使うことで、同じ日は同じ結果になる（日替わり感の演出）
- ただし「もう1本引く」時は乱数を追加して毎回変わるようにする
