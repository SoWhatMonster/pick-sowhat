## 概要

毎日AIが自動生成する「今日のおすすめ10本」を表示するセクションを追加する。日付をシードにしてサーバー側でキャッシュし、同じ日は同じリストを返す。

---

## 配置場所

FAQの直前（Phase 1コンテンツの後）。

```
[既存] ヒーロー・診断ツール
[Phase 1] 産地ガイド・シーン別特集
[Phase 2] おみくじ
[Phase 3] 今日のおすすめ10本  ← 追加
[既存] FAQ
```

## UI仕様

### セクション見出し

```
今日のおすすめ
```
英語サブ:
```
Today's picks. Updated daily.
```

日付表示（例）:
```
2026年3月21日（土）
```

### リスト表示

- 10枚のカード横スクロール（モバイル）/ 5列×2行グリッド（PC）
- 各カード:
  - 順位番号（01〜10）
  - ボトル画像プレースホルダー（Amazon PA-API、または SVGフォールバック）
  - 銘柄名
  - タグ2〜3個
  - Amazon・楽天ボタン

## APIとキャッシュ仕様

### キャッシュ戦略

- Next.jsの `revalidate` を使い、**1日1回だけAPIをコール**してキャッシュする
- キャッシュキー: `daily-picks-YYYY-MM-DD`
- Vercel KVまたはVercel Postgresに保存

```ts
// app/api/daily-picks/route.ts
export const revalidate = 86400 // 24時間

export async function GET() {
  const today = new Date().toISOString().split('T')[0]
  // キャッシュ確認 → なければAI生成 → 保存 → 返す
}
```
